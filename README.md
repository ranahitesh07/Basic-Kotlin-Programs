# Basic Kotlin Programs

This repository contains basic Kotlin programming assignments.

## Questions Included

1. Print numbers from 1 to 10
2. Check even or odd number
3. Sum of numbers from 1 to N
4. Reverse a number
5. Palindrome number check
6. Factorial using loop
7. Largest among three numbers
8. Count digits in a number
9. Multiplication table
10. Vowel or consonant check
11. Sum of digits
12. Swap two numbers
13. Simple interest calculation
14. Prime numbers from 1 to N
15. Fibonacci series
16. Temperature conversion

## Technologies Used

- Kotlin
- GitHub
- VS Code / Kotlin Playground
